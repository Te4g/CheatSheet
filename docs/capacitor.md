**Add to existing app**
```shell script
npm install --save @capacitor/core @capacitor/cli
```
**Then, initialize Capacitor with your app information.**
```shell script
$ npx cap init
```
**Build beforce lauching capacitor**
```shell script
$ yarn build
```
**Next, install any of the desired native platforms:**
```shell script
$ npx cap add android
$ npx cap add ios
$ npx cap add electron
```
